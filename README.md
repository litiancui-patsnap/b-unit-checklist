# 每日英语学习打卡

> 一个帮助用户每天完成英语输入、跟读和输出的小程序。

## 项目简介

这是一个轻量的英语学习陪伴小程序，适合想每天坚持学英语的用户。用户首次进入时选择学习目标和每日强度，小程序会生成对应的今日学习计划，通过英语任务、英文日记模板、学习记录和分享文案，帮助用户形成稳定习惯。

## 核心功能

### 1. 今日英语任务
- 首次使用引导选择学习目标和每日强度
- 根据学习目标生成今日英语任务
- 支持轻量、标准、强化三档任务
- 展示预计学习时间和模块分布
- 实时统计今日完成进度
- 记录连续学习天数
- 查看最近 7 天学习情况

### 2. 英文日记一句话
- 根据学习目标提供英文日记模板
- 点选模板后快速改写
- 保存到当日学习记录
- 完成打卡后自动进入分享文案

### 3. 英语设置
- 选择学习目标：日常英语、口语表达、四六级备考、考研英语、职场英语
- 设置每日默认强度
- 一键按当前目标生成默认任务
- 自定义学习前准备
- 自定义轻量、标准、强化任务
- 设置每日学习提醒

### 4. 学习记录 / 周报
- 导出最近 30 天英语学习记录
- 统计本周完成天数和英文日记数量
- 支持仅导出完成日
- 一键复制到微信、备忘录或社群

## 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序账号，测试号也可

### 运行方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 选择当前项目目录
4. 填写 AppID 或使用测试号
5. 编译运行

## 数据存储

所有数据存储在本地 `wx.Storage`：
- 用户配置
- 每日英语学习记录
- 英文日记
- 提醒元数据

当前版本会把旧版默认配置迁移为英语学习配置，但会保留历史天记录。

## 技术栈

- 微信小程序原生框架
- JavaScript + WXML + WXSS
- `wx.Storage` 本地存储

## 默认任务设计

默认任务会根据学习目标变化：

- 日常英语：单词、听力、跟读、英文日记
- 口语表达：真实口语音频、跟读、复述、口语脚本
- 四六级备考：核心词、听力小题、阅读材料、作文表达
- 考研英语：核心词、长难句、翻译、同义改写
- 职场英语：会议表达、邮件句子、职场音频、模拟发言

每个目标都提供轻量、标准、强化三档任务，并标记模块和预计分钟数。
