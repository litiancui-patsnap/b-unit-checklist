# 语言学习计划与英语执行

> 先规划今天学什么，再在小程序内完成学习。

## 项目简介

小程序由“计划首页”和“学习执行页”组成。首页按星期生成日语、英语与复习任务，展示时间投入、周节奏、语言比例和对应资料；英语任务可以直接进入执行页，使用每日内容、单词、复习、小测、跟读和英文日记等学习工具。

## 核心功能

### 1. 语言学习计划
- 按周一至周日生成不同学习重点与任务组合
- 支持日期切换和本周完成节奏
- 展示计划时长、完成时长、连续打卡和语言比例
- 每项任务包含语言、时长和对应学习资料
- 支持添加和删除当天自定义任务
- 英语任务可跳转到小程序内的对应学习模块

### 2. 学习执行页
- 每日英语句子、短文、场景表达和长难句
- 今日单词本、到期复习和单词小测
- 场景口语、发音播放和英文日记
- 支持轻量补救、完成打卡、分享和最近 7 天记录

### 3. 英文日记一句话
- 根据学习目标提供英文日记模板
- 点选模板后快速改写
- 保存到当日学习记录
- 完成打卡后自动进入分享文案

### 4. 英语设置
- 选择学习目标：日常英语、口语表达、四六级备考、考研英语、职场英语
- 设置每日默认强度
- 一键按当前目标生成默认任务
- 自定义学习前准备
- 自定义轻量、标准、强化任务
- 设置每日学习提醒

### 5. 学习记录 / 周报
- 导出最近 30 天英语学习记录
- 统计本周完成天数和英文日记数量
- 支持仅导出完成日
- 一键复制到微信、备忘录或社群

## 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序账号，测试号也可

### 运行方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 选择当前项目目录
4. 填写 AppID 或使用测试号
5. 编译运行

## 数据存储

所有数据存储在本地 `wx.Storage`：
- 用户配置
- 每日计划任务、完成状态和自定义任务
- 每日英语学习记录
- 英文日记
- 提醒元数据

当前版本会把旧版默认配置迁移为英语学习配置，但会保留历史天记录。

## 技术栈

- 微信小程序原生框架
- JavaScript + WXML + WXSS
- `wx.Storage` 本地存储

## 默认任务设计

默认任务会根据学习目标变化：

- 日常英语：单词、听力、跟读、英文日记
- 口语表达：真实口语音频、跟读、复述、口语脚本
- 四六级备考：核心词、听力小题、阅读材料、作文表达
- 考研英语：核心词、长难句、翻译、同义改写
- 职场英语：会议表达、邮件句子、职场音频、模拟发言

每个目标都提供轻量、标准、强化三档任务，并标记模块和预计分钟数。
