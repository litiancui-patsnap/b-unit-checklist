# B类备考打勾清单

> 江苏事业单位 B 类备考专用打卡小程序

## 项目简介

这是一个专为江苏事业单位 B 类考试备考者设计的微信小程序，帮助在职备考者每天按疲劳程度选择不同强度的学习任务清单，养成良好的备考习惯。

## 核心功能

### 1. 今日打卡
- **启动流程**：进入学习状态的仪式感启动步骤
- **ABC 模板**：根据疲劳程度选择不同强度的学习计划
  - A 模板：低强度（适合疲惫时）
  - B 模板：中强度（正常状态）
  - C 模板：高强度（精力充沛时）
- **进度统计**：实时显示今日完成进度
- **连续打卡**：记录连续完成天数
- **最近 7 天**：快速查看近期打卡情况

### 2. 自定义设置
- **启动流程配置**：自定义学习前的准备步骤
- **ABC 模板配置**：
  - 自定义每个模板的任务内容
  - 设置完成阈值（需勾选多少项才算完成）
- **提醒功能**：
  - 开启/关闭每日提醒
  - 自定义提醒时间（默认 21:30）
- **一键恢复默认**：快速恢复出厂设置

### 3. 数据导出
- **30 天记录导出**：导出最近 30 天的打卡记录
- **一键复制**：快速复制导出文本
- **筛选功能**：可选择仅导出已完成日期

## 使用说明

### 首次使用

1. 打开小程序，进入"设置"页
2. 根据个人需求自定义启动流程和 ABC 模板
3. 点击"保存配置"

### 日常使用

1. 每天打开小程序
2. 根据当天疲劳程度选择 A/B/C 模板
3. 按照启动流程进入学习状态
4. 完成任务后勾选清单
5. 达到阈值后点击"完成今日"

### 数据管理

- **查看历史**：在今日页底部查看最近 7 天记录
- **导出备份**：进入"导出"页，一键复制最近 30 天记录
- **重置数据**：今日页可重置当天数据，设置页可恢复默认配置

## 数据存储

所有数据存储在本地 Storage，包括：
- 用户自定义配置（启动流程、ABC 模板、提醒设置）
- 每日打卡记录（选择的模板、勾选情况、完成状态）
- 提醒元数据（上次提醒日期）

**数据格式**：
```json
{
  "config": {
    "startChecklist": [...],
    "templates": { "A": {...}, "B": {...}, "C": {...} },
    "reminder": { "enabled": false, "time": "21:30" }
  },
  "days": {
    "2025-12-23": {
      "template": "B",
      "start": {...},
      "items": {...},
      "complete": true
    }
  },
  "meta": {
    "remind_last_shown_date": "2025-12-23"
  }
}
```

## 最佳实践

### 启动流程建议
- 手机静音并放远处
- 准备一杯水
- 打开计时器
- 深呼吸 3 次

### 模板设置建议

**A 模板（低强度）**：
- 完成阈值：1 项
- 适合：工作日晚上疲惫时
- 内容：简单任务、复习笔记

**B 模板（中强度）**：
- 完成阈值：2 项
- 适合：周末或精力正常时
- 内容：刷题、看视频、整理错题

**C 模板（高强度）**：
- 完成阈值：2-3 项
- 适合：假期或精力充沛时
- 内容：模拟考试、专项训练、大量刷题

## 常见问题

**Q: 数据会丢失吗？**
A: 数据存储在微信本地，除非卸载小程序或清除缓存，否则不会丢失。建议定期导出备份。

**Q: 可以修改历史数据吗？**
A: 目前只能重置当天数据，历史数据不可修改（防止作弊）。

**Q: 提醒功能如何工作？**
A: 当开启提醒后，在设置的时间之后打开小程序，如果当天未完成会弹出提醒（每天仅提醒一次）。

**Q: 可以同时使用多个设备吗？**
A: 数据存储在本地，不同设备数据独立，暂不支持云同步。
