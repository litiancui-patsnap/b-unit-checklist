# 每日英语学习打卡

> 一个帮助用户每天完成英语输入、跟读和输出的小程序。

## 项目简介

这是一个轻量的英语学习陪伴小程序，适合想每天坚持学英语的用户。用户可以选择学习目标和每日强度，通过学习前准备、英语任务清单、英文日记和学习记录，形成稳定的英语学习习惯。

## 核心功能

### 1. 今日英语任务
- 根据默认强度展示今日英语任务
- 支持轻量、标准、强化三档任务
- 实时统计今日完成进度
- 记录连续学习天数
- 查看最近 7 天学习情况

### 2. 英文日记一句话
- 每天写一句英文
- 保存到当日学习记录
- 完成打卡后自动进入分享文案

### 3. 英语设置
- 选择学习目标：日常英语、口语表达、四六级备考、考研英语、职场英语
- 设置每日默认强度
- 自定义学习前准备
- 自定义轻量、标准、强化任务
- 设置每日学习提醒

### 4. 学习记录 / 周报
- 导出最近 30 天英语学习记录
- 统计本周完成天数和英文日记数量
- 支持仅导出完成日
- 一键复制到微信、备忘录或社群

## 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序账号，测试号也可

### 运行方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 选择当前项目目录
4. 填写 AppID 或使用测试号
5. 编译运行

## 数据存储

所有数据存储在本地 `wx.Storage`：
- 用户配置
- 每日英语学习记录
- 英文日记
- 提醒元数据

当前版本会把旧版默认配置迁移为英语学习配置，但会保留历史天记录。

## 技术栈

- 微信小程序原生框架
- JavaScript + WXML + WXSS
- `wx.Storage` 本地存储

## 默认任务设计

### 轻量
- 背 10 个单词
- 听 5 分钟英语音频
- 跟读 3 句英文

### 标准
- 背 20 个单词并复习昨日单词
- 听 10 分钟英语音频
- 跟读 5 句英文
- 写 1 句英文日记

### 强化
- 背 40 个单词并整理易错词
- 精听 15 分钟英语音频
- 跟读 10 句英文并录音回听
- 阅读 1 段英文短文
- 写 3 句英文日记
