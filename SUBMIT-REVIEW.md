# 小程序提交审核指南

## 📋 审核前检查清单

### ✅ 必须完成的配置

- [ ] **服务器域名白名单已配置**
  - 登录：https://mp.weixin.qq.com/
  - 路径：开发 → 开发管理 → 开发设置 → 服务器域名
  - 添加到 request合法域名：`https://b-unit-checklist-main.vercel.app`

- [ ] **Vercel 环境变量已配置**
  - 访问：https://vercel.com/dashboard
  - 确认 `DASHSCOPE_API_KEY` 已配置

- [ ] **功能测试通过**
  - 查词功能正常
  - 学习内容生成正常
  - 学习计划生成正常
  - 所有页面正常跳转

- [ ] **隐私协议和用户协议**（如果有）
  - 确认隐私政策链接有效
  - 确认用户协议链接有效

## 📤 提交审核步骤

### 1. 打开微信开发者工具
- 打开项目：`/Users/ltc/Documents/project/b-unit-checklist-main`

### 2. 上传代码
1. 点击工具栏：**上传**
2. 填写版本号，例如：`1.0.0`
3. 填写项目备注，例如：
   ```
   迁移到 Vercel API，新增英语学习功能
   - 修复按钮布局问题
   - 优化学习体验
   - 性能优化
   ```
4. 点击**上传**

### 3. 登录微信公众平台提交审核
1. 访问：https://mp.weixin.qq.com/
2. 进入：**版本管理**
3. 找到刚上传的版本（开发版本）
4. 点击：**提交审核**

### 4. 填写审核信息

#### 基本信息
- **小程序名称**：英语学习打卡小程序（或你的实际名称）
- **小程序简介**：每日英语学习打卡工具，支持单词、听力、口语、阅读、写作等多种学习方式

#### 服务类目
建议选择：
- **教育 → 在线教育**
- 或 **工具 → 效率工具**

#### 审核页面配置
需要配置审核入口页面：
1. **首页**：`pages/onboarding/onboarding` 或 `pages/home/home`
   - 页面功能说明：学习任务打卡页面
   - 测试账号：无需（不涉及登录）

2. **其他页面**（如果要求）：
   - `pages/settings/settings` - 设置页面
   - `pages/export/export` - 学习记录页面

#### 测试账号
- 如果不需要登录：选择**无需测试账号**
- 如果需要登录：提供测试账号和密码

### 5. 确认提交
- 仔细检查所有信息
- 确认服务器域名已配置
- 点击**提交审核**

## 📝 审核注意事项

### 容易被拒的原因
1. **服务器域名未配置** - 最常见
2. **功能描述不清晰**
3. **隐私政策缺失**（如果收集用户信息）
4. **服务类目选择错误**
5. **测试路径不完整**

### 审核时间
- 通常：1-7 个工作日
- 加急审核：需要特殊申请

### 审核失败怎么办
1. 查看拒绝原因
2. 修改代码或配置
3. 重新上传和提交

## 🔍 审核状态查询

登录微信公众平台查看：
- **版本管理 → 审核版本**
- 状态：审核中 / 审核通过 / 审核失败

## 📱 审核通过后

1. **发布上线**
   - 点击**提交发布**
   - 等待发布完成（通常几分钟）

2. **生成体验版二维码**
   - 可以先生成体验版给用户测试
   - 开发管理 → 开发版本 → 体验版

3. **监控线上问题**
   - 查看运营数据
   - 关注用户反馈
   - 查看 Vercel 日志

## 🆘 常见问题

**Q: 提示"不在以下 request 合法域名列表中"**
A: 必须在微信公众平台配置服务器域名白名单

**Q: 审核时测试人员无法使用功能**
A: 确保：
- 服务器域名已配置
- Vercel API 正常运行
- 提供清晰的功能说明

**Q: 如何加急审核？**
A: 微信公众平台 → 版本管理 → 加急审核（需满足条件）

**Q: 审核被拒后多久可以再提交？**
A: 立即可以重新提交，但要先修复问题
